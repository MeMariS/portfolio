1. Чтобы создать страницу, создаем новый файл с расширением HTML (напр., work.html)
2. CSS код разделяем по файлам: отдельный файл для общих стилей на все страницы (global.css), отдельный – для каждой страницы
3. Шрифты: чтобы они работали, нужно:
  a. Подключить шрифт (через <link> в html-файл или через @import в css-файл)
  b. Применить шрифт (через font-family: "название шрифта")
